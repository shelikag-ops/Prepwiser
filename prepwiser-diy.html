    Google Sheets Sync

    Connect to a Google Sheet to sync your meal plans, recipes, shopping list, and pantry across all your devices. Changes are saved locally first, then synced to your Sheet when connected.

    1. Setup Instructions (one-time)

    1. Go to Google Cloud Console and create a project (or use an existing one)
    2. Enable the Google Sheets API (APIs & Services → Library → search "Sheets")
    3. Create an OAuth 2.0 Client ID:
      Credentials → Create Credentials → OAuth client ID → Web application
      Add your domain to Authorized JavaScript origins (for local files, use http://localhost)
    4. Also create an API Key (Credentials → Create Credentials → API Key)
    5. Create a new Google Sheet and grab the Sheet ID from the URL:
      https://docs.google.com/spreadsheets/d/{SHEET_ID}/edit
